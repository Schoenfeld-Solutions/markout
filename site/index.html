Markdown for Outlook

Schoenfeld Solutions

Safe Markdown authoring for Outlook compose.

MarkOut is an Outlook compose add-in that turns Markdown into inline-styled email HTML, keeps rendering safe, and supports manual rendering plus Smart Alerts auto-rendering before send.

Install

Sideload via Outlook Add from File.

1

Open Outlook, then go to Get Add-ins and choose Add a custom add-in.

2

Choose Add from File. Microsoft no longer supports Add from URL for this workflow.

3

Use manifest.beta.xml for staged validation or manifest.xml for the production deployment. If a browser opens XML inline, use the download buttons on this page or save the linked file before selecting it in Outlook.
