Install

Sideload with Add from File

Outlook no longer supports Add from URL for this workflow. Use the custom add-in file flow instead.

  1. 1

    Open the Custom Add-in Flow

    In Outlook, open Get Add-ins and choose Add a custom add-in.

  2. 2

    Choose Add from File

    Select Add from File and point Outlook at the manifest you want to test.

  3. 3

    Prefer Beta for Validation

    Use manifest.beta.xml for current staged changes and manifest.xml for the stable hosted deployment.
